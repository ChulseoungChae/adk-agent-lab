# 1장 실시간 이상감지: Rule 기반 이상 패턴 분석

## 1.1 Rule 기반 이상감지의 원리

반도체 공정에서는 실시간으로 수집된 공정 파라미터 값이 사전에 정의된 상한선(Upper Limit) 및 하한선(Lower Limit)을 초과하거나 미만일 때 이상으로 판단합니다. 이러한 Rule 기반 이상감지는 실시간 로그(index4)를 통해 지속적으로 모니터링되며, 공정 중 즉시 대응이 가능합니다.

## 1.2 실제 이상 사례: MFC8_NH3 상한선 초과

실제 이상감지 로그를 분석한 결과, **MFC8_NH3** 파라미터의 상한선 위 violaton이 다수 확인되었습니다. MFC8_NH3은 ALD/PVD 공정에서 암모니아(NH₃) 가스를 공급하는 Mass Flow Controller의 설정값으로, 정상 범위는 -0.2 ~ 4.7입니다.

## 1.3 이상 발생의 공정 영향

MFC8_NH3이 상한선을 초과할 경우, NH₃ 가스 유량이 과다 공급되어 ALD 공정의막 두께 불균일, 파티클 발생, 레시피 단계(step) 중단(is_interrupted) 등이 발생할 수 있습니다. 위 사례에서 is_interrupted 값이 1 또는 2로 기록된 것은 공정 중단의 심각도를 나타냅니다.

## 1.4 Rule 기반 이상감지의 한계와 활용

Rule 기반 이상감지는 단순하지만 실시간 대응이 가능하며, 설정된 임계값을 즉시 위반할 때 경고합니다. 그러나 과도한 False Positive를 유발할 수 있으므로, 예측 모델 기반 이상감지(2장)와 함께 활용하여 정밀도를 보완해야 합니다.
