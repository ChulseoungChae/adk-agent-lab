# 4. 리포트 기반 통계 분석: 일간·월간 이상감지 집계

# 4장 리포트 기반 통계 분석: 일간·월간 이상감지 집계

## 4.1 리포트 시스템의 역할

플랫폼은 자동 생성된 일간(Daily) 및 월간(Monthly) 이상감지 리포트를 제공합니다. 이 리포트들은 실시간 이상감지 및 예측 이상감지 데이터를 집계하여, 엔지니어가 주기적으로 공정 상태를 한눈에 파악할 수 있도록 합니다.

## 4.2 리포트 목록 현황

현재 플랫폼에는 다음과 같은 리포트가 저장되어 있습니다.

### 월간 리포트
| 리포트 ID | 기간 | 생성일 |
|-----------|------|--------|
| report_monthly_2026-01 | 2026-01-01 ~ 2026-02-01 | 2026-02-24 |
| report_monthly_2025-12 | 2025-12-01 ~ 2026-01-01 | 2026-02-24 |
| report_monthly_2025-11 | 2025-11-01 ~ 2025-12-01 | 2026-02-24 |

### 일간 리포트 (일부)
| 리포트 ID | 기간 | 생성일 |
|-----------|------|--------|
| report_daily_2026-02-23 | 2026-02-23 | 2026-02-24 |
| report_daily_2026-02-22 | 2026-02-22 | 2026-02-24 |

총 117건의 리포트가 생성되어 있으며, 일간 리포트는 매일 자정 전에 자동으로 생성됩니다.

## 4.3 리포트 콘텐츠 구성

리포트는 다음과 같은 통계 정보를 포함합니다:
- **이상감지 건수**: 실시간 Rule 기반 이상과 예측 모델 이상 분리 집계
- **파라미터별 이상 발생 순위**: MFC8_NH3, MFC3_N2-3 등 주요 이상 파라미터
- **단계(step)별 이상 분포**: N-FL1, pre-NH3P 등 공정 단계별 이상 발생 패턴
- **일일 이상 추이**: 시간별 이상 발생 건수 그래프 데이터
- **이상 유형별 분류**: 상한 초과, 하한 초과, 예측치 편차 등

## 4.4 리포트의 실무 활용

### 일간 리포트 활용
- 전일 공정 상태 빠르게 확인
- 주요 이상 파라미터 식별
- 다음 day 공정 준비 (예: MFC8_NH3 이상 다발 시 예비 부품 확인)

### 월간 리포트 활용
- 월간 공정 안정성 지표 산정
- 레시피별/장비별 이상률 비교
- 공정 개선 계획 수립 (예: MFC8_NH3의 지속적 상한 초과 → 유량 교정 주기 단축)

## 4.5 리포트 자동 생성 워크플로우

1. **데이터 수집**: 생성기(Genertor)가 실시간으로 공정 데이터 수집 (RUN 상태 확인 필요)
2. **이상 감지**: Rule 기반 및 PatchTST 예측 모델이 이상 판단
3. **집계**: 일간/월간 단위로 이상 건수, 파라미터, 단계별 통계 집계
4. **리포팅**: JSON 파일로 저장 및 정적 페이지 제공
5. **확인**: 엔지니어가 리포트 확인 후 조치

이러한 워크플로우 덕분에 엔지니어는 수동으로 이상 데이터를 수집·집계할 필요 없이, 플랫폼이 자동 생성한 리포트를 통해 공정 상태를 모니터링할 수 있습니다.
