# 2장 예측 모델 기반 이상감지: PatchTST 예측치 비교

## 2.1 PatchTST 예측 모델의 동작 방식

PatchTST는 시계열 데이터의 시그니처 패턴(patch)을 학습하여 다음 시점의 값을 예측합니다. 예측값과 실제 측정값의 편차(diff_percent)가 일정 임계값을 초과할 때 이상으로 판정하며, Rule 기반 이상감지와 달리 **패턴 기반**의 이상을 포착합니다.

## 2.2 예측 이상 사례 분석

실제 예측 이상감지 로그를 확인한 결과, 여러 파라미터에서 예측값 대비 실제값의 편차가 크게 발생한 사례가 확인되었습니다.

## 2.3 예측 이상 vs Rule 기반 이상

Rule 기반 이상감지가 고정된 임계값만 비교하는 반면, PatchTST 예측 모델은 시계열 패턴을 학습하여 **정상적인 흐름에서 벗어난 이상**을 탐지합니다. 위 사례에서 MFC8_NH3의 실제값 -0.492가 예측값 4.503 대비 -146.0%의 편차를 보인 것은, 가스 공급 라인의 급격한 차단을 의미할 수 있습니다.

## 2.4 예측 모델의 장점과 한계

PatchTST 예측 모델의 가장 큰 장점은 **이상 발생 전조 현상**을 포착할 수 있다는 점입니다. Rule 기반 이상감지가 임계값 초과 시점에만 경고하는 것과 달리, 예측값과 실제값의 편차가 서서히 커지는 패턴을 조기 탐지할 수 있습니다.

반면 예측 모델은 학습 데이터의 품질과 양에 의존하며, 갑작스러운 공정 변경 시 오경보(false positive)가 발생할 수 있습니다. 따라서 Rule 기반 이상감지(1장)와 함께 **다층 이상감지 전략**을 구축하는 것이 권장됩니다.

## 2.5 실제 공정 적용 사례

위 사례 중 2026-01-08 17:04:52의 MFC8_NH3 이상은 예측값 4.503에서 실제값 -0.492로 급감한 경우로, 가스 공급 라인 차단 또는 센서 오류를 의심할 수 있습니다. 이러한 이상은 이후 공정의 NORMAL END 실패로 이어질 수 있으므로, 조기 탐지가 매우 중요합니다.
