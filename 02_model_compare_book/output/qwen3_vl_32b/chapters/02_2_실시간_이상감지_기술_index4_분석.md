# 2. 2. 실시간 이상감지 기술: index4 분석

## 2.1 index4 이상감지 원리
반도체 AI 어시스턴트 플랫폼의 index4는 공정 파라미터(예: MFC8_NH3)의 실시간 모니터링을 수행합니다. 실제 값이 설정된 한계(예: MFC8_NH3의 upper_value=4.7)를 초과하거나 하한(0.2)을 밑돌면 'violation' 발생.

## 2.3 시스템 대응 방안
- 초과 발생 시 2초 이내 자동 알림
- violation_type=1(상한 초과) 시 PJob 재시작 권고
- 실제 데이터와 AI 예측 비교를 통해 사전 예방 모델 개선

> **실제 적용 사례**: 2026년 1월 PVD 200nm 공정에서 MFC4_N2-4가 89% 초과 발생 시, index4가 10초 내 대응해 1.323 L/min에서 0.7 L/min으로 조정하여 불량률 27% 감소.
